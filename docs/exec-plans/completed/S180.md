# Exec Plan: S180 -- Chat interface simplification: settings drawer for model/web toggle, collapsible transparency panel

**Phase:** V3 Phase 3 - UX Polish and Learner Focus
**Priority:** 20
**Status:** in-progress

## Goal

Move the LLM model selector and web search toggle out of the Chat tab header into a right-side settings drawer (Sheet component), reducing header clutter so document context chips and conversation controls are the primary affordances. The 'How I answered' transparency panel is refactored to be collapsed by default -- showing only a small 'i' icon -- and expanded per-message on click. Low-confidence indicators shrink from full-width banners to compact inline badges next to the message timestamp area. The ConfusionBanner (flashcard nudge) remains full-width as it requires user action.
