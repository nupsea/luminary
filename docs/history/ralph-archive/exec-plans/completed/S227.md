# Exec Plan: S227 -- Rename Evals to Quality + tabbed hub (Datasets, Runs, Routing, Ablations, Regressions)

**Phase:** V3 Phase 8 - Evaluation and Correctness
**Priority:** 67
**Status:** in-progress

## Goal

Rename the Evals navigation tab to "Quality" and restructure the page as a five-sub-tab hub (Datasets, Runs, Routing, Ablations, Regressions). File-backed golden dataset cards (source=file) are currently non-clickable; this story makes them fully interactive with a detail panel, paginated question preview from JSONL, past-runs table, and a Run Eval trigger. Three new backend endpoints are added to evals.py to power the new surfaces. No backend route renames, no schema changes.
