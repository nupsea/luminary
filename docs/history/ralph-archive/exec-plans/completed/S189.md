# Exec Plan: S189 -- Auto-organize: guided plan with confirmation workflow

**Phase:** V3 Phase 4 - Learner Experience Refinement
**Priority:** 29
**Status:** in-progress

## Goal

Replace the existing individual card-by-card accept/reject UI for cluster suggestions with a holistic Organization Plan dialog. After POST /notes/cluster completes, the frontend shows a full-page dialog presenting all pending suggestions as an editable plan. The user can rename groups, drag notes between them, include/exclude groups, and click "Apply Plan" to create all selected collections in a single backend transaction. This reduces friction and gives users a complete picture of how their notes would be reorganized before committing.
