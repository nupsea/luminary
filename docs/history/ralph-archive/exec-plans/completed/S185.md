# Exec Plan: S185 -- Simplified Study tab: single generate button with adaptive defaults

**Phase:** V3 Phase 4 - Learner Experience Refinement
**Priority:** 25
**Status:** in-progress

## Goal

S185 is a pure frontend refactor. The Study tab currently renders several competing panels -- a SmartGeneratePanel that wraps GeneratePanel (with its three sub-tabs for Basic, Graph, and Cloze), a DeckStatusAccordion (already merging DeckHealthPanel + HealthReportPanel), and a StrugglingPanel -- along with GoalsPanel (already removed from Study tab per S177 but referenced here for clarity). The goal is to tighten the layout: two CTAs at the top (Start Studying and Generate Cards), the S184 search/filter card list as the primary visible area, and all health/insight panels collapsed into a single Insights accordion below the card list. No backend changes are needed; all existing API endpoints are reused.
