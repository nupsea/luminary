# Knowledge Model / Knowledge Universe -- ABANDONED

This direction (a goal-driven "curriculum spine" plus a rendered **Knowledge Universe** of
auto-clustered galaxies) was a **failed experiment** and was removed on 2026-06-23.

Do not implement from it. Do not restore the Universe page, the Goals/curriculum nav tab, or the
`curriculum` router/service/models. The concept graph, topics, and mastery substrate were kept
because the core Study + Ask flow depends on them; everything specific to the Universe/curriculum
projection is gone.

**Why it failed:** the map was not the moat. An auto-clustered knowledge map is a demo anyone can
copy, and the goal/curriculum spine added a heavy layer users did not need. The locked product
thesis stands: the moat is source-grounded measurement + a compounding learner record (FSRS +
calibration), not content generation or a graph visualization.

The full original 274-line spec and the deleted `universe.md` / `knowledge-universe-roadmap.md`
remain in git history (branch `feat/knowledge-universe`, commits up to `418c5f8`) if anyone ever
needs the detail. There is no reason to read them for current work.
