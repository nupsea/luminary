# Security Policy

## Reporting a Vulnerability

We take the security of Learning Mate seriously. If you find a security vulnerability, please do not open an issue on the repository. Instead, please report it privately.

Please send an email to [sethurama@example.com] (replace with actual email) with details about the vulnerability. We will acknowledge your report within 48 hours and provide a timeline for a fix.

Thank you for helping keep Learning Mate secure!
